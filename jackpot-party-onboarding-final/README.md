# Jackpot Party Onboarding Hub

This package matches the latest themed preview and is ready for Vercel.

## Deploy on Vercel
1. Unzip this project
2. Create a new GitHub repo (private is fine)
3. Upload these files to the repo
4. In Vercel, create a new project and import that repo
5. Vercel should detect it automatically
6. Use:
   - Framework: Vite
   - Build command: npm run build
   - Output directory: dist

## Local run
npm install
npm run dev

## Notes
- Demo data persists in browser localStorage
- Update names, links, and content before broad sharing
